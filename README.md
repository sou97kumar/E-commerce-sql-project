# 🛒 E-Commerce SQL Project

## 📌 Overview

This project demonstrates a basic e-commerce database system using SQL. It includes customers, products, orders, and order items.

## 🧱 Database Schema

* **customers**
* **products**
* **orders**
* **order_item**

## 🔗 Relationships

* Customers → Orders (1)
* Orders → Order Items (1)
* Products → Order Items (1)

## ⚙️ Features

* Primary & Foreign Keys
* Joins
* Aggregations
* Real-world dataset simulation

## 📊 Sample Queries

* Total revenue calculation
* Top-selling products
* Customer order history

## 🚀 How to Run

1. Run `schema.sql`
2. Run `data.sql`
3. Run `queries.sql`
